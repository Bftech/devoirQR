Le QR code : origines et évolutions

Le QR code est un moyen de stockage de l'information sous forme graphique à 2 dimensions. Il peut stocker une grande variété de données tel que du texte, des chiffres, des URLs, mais aussi des contacts, déclencher des actions (appels, sms, mails) ou représenter un point géographique.

Il à été inventé en 1994 par une entreprise Japonaise pour les usines Toyota, puis rendu public en 1999 sous licence libre. Ce n'est qu'avec l'arrivée des smartphones et leurs caméras embarquées, que le QR code se répand vraiment.

Depuis les normes se suivent et ajoutent de nouvelles fonctionnalités au QR code. (la dernière norme ISO date de 2015)

Fonctionnement du QR code

Pour pouvoir fonctionner efficacement, le QR code est structuré de façon bien particulière :

  • Des cibles de positionnement
  • Un ou plusieurs indicateurs d'alignement
  • Des symboles de synchronisation
  • Les informations de version
  • Les informations de format

Grâce aux cibles et aux indicateurs, le code est lisible sous de nombreux angles, facilitant ses possibilités d'implantations. Ensuite les symboles de synchronisation et les informations de version et de format définissent la manière de lire le code.

Avantages / Inconvénients

Avantages

  • Rapide à scanner
  • Peu de prérequis pour le lire, juste une caméra
  • L'information redondante et le système de correction d'erreurs permettent de lire un code dégradé
  • Personnalisable

Inconvénients

  • Peu atteindre de très grandes tailles si beaucoup d'informations
